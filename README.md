# Room Condition Monitoring System
Originally developed in 2020 as part of Embedded System Design Course; this repository contains the finalized and cleaned version for public release.
## Overview
This project implements an embedded monitoring system designed to measure room temperature and ambient light intensity, with automatic alarm activation when user-defined thresholds are exceeded. The system enhances environmental awareness and safety by providing real-time feedback and alerts.
